# Echoes of Slime

> *Un jeu de plateforme-puzzle où vous incarnez une slime capable de moduler sa masse pour résoudre des énigmes et traverser des environnements hostiles.*

---

## Description

Dans **Echoes of Slime**, vous contrôlez une créature gluante dotée d'une capacité unique : absorber, déposer et redistribuer sa propre matière. Plus vous êtes lourd, plus vous êtes puissant — mais moins vous êtes agile. Trouvez le bon équilibre pour franchir les obstacles, activer les mécanismes et atteindre la sortie.

- **Trois états** : normal, lourd et léger, chacun avec ses propres capacités
- **Environnements variés** : plateformes, piques, caisses, ventilations, portes à activer
- **Mécanique de slime** : déposez des morceaux de vous-même pour résoudre les puzzles

---

## Contrôles

| Touche | Action |
|---|---|
| `Q` / `D` | Se déplacer |
| `Espace` | Sauter (double saut disponible) |
| `S` | Dash |
| `LCtrl` | Activer / désactiver la course |
| `A` | Déposer un morceau de slime |
| `W` | Absorber un morceau de slime |
| `Échap` | Pause |
